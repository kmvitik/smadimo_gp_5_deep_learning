# Прогнозирование РТО с помощью табличных данных и MLP

## Описание
В этой части решалась задача прогнозирования РТО магазинов x5 group. Из датасета использовались данные о характеристиках магазинов, городе, регионе, потоке клиентов, окружении, количестве касс и других факторах

Целевая переменная: `РТО`.
- Задача регрессии
- Метрика `MAPE`

## Файлы
- `eda_MLP.ipynb` - разведочный анализ данных, визуализации, построение baseline-моделей и MLP

- `train_2.csv` - исходный датачсет
