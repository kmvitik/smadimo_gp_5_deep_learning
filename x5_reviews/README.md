## Задача

Многометочная классификации отзывов о продуктовых магазинах.

## Файлы

- `*.csv` - датасеты
- `reviews.ipynb` — первичный анализ исходного датасета.
- `llm_label.ipynb` — разметка проблем с помощью LLM.
- `prepare.ipynb` — подготовка и проверка датасета.
- `bert.ipynb` — обучение модели RuBERT.
- `models/` — директория для сохранения моделей.

## Целевые метки

Модель предсказывает несколько типов проблем в отзыве:

- `product_quality` - качество продуктов
- `staff_service` - плохой персонал
- `queues_checkout` - очереди
- `assortment_availability` - плохой ассортимент
- `store_cleanliness` - чистота
- `other_problem` - прочее
