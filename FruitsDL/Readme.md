# Классификация фруктов и овощей для весовых аппаратов 

## Что делалось: 
Решалась задача классифиуации для фруктов и овощей. Решение может использоваться для задач в ритейле на кассах самообслуживания, весовых аппаратов для корректного выбора покупаемого продукта и не только.
Для решения задачи использовался датасет fruits-360_100x100. Из 257 классов модели обучались и сравнивались на одих и тех же 36 классах, выбраны не все классы ввиду специфичности продуктов в датасете. 

## Данные 
Датасет - fruits-360
- `Training` - для обучения и 15% в валидацию
- `Test` - для проверки качества

Для ускорения обучения ограничил изображения, подающиеся на класс. Это 200 на train и 80 на тест 

## Модели 
Были обучены и сравнивалсь четыре архитектуры

- `FirstCNN` - базовая сверточная нейронная сеть
- `SecondCNN` - сверточная сеть с BatchNorm и Dropout
- `AlexNet` - предобученная модель
- `EfficientNet_B0` - предобученная модель

## Для логирования использовался Wandb, все файлы с весами и распределение train/val/test сохраняются тоже туда
